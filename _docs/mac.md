## [mac|zip] MacでコマンドでZIP圧縮する
e.g. DS_Store を含めないで圧縮する
```
$ zip -r hogehoge.zip * -x *.DS_Store
```

暗号化ZIPにする（eオプションを付ける）
```
$ zip -er hogehoge.zip * -x *.DS_Store
Enter password:
Verify password:
```
