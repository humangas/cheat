    humangas's cheat sheet¶

    See also: github.com/humangas/cheat

    aws/iam¶

    • IAMユーザー一覧取得する

    aws/s3¶

    • S3バケット一覧を取得する

    du¶

    • ディレクトリ単位の容量 主要オプションが知りたい
    • デカ目のファイルの合計を降順ソートして表示したい
    • ファイルサイズランキング TOP10
    • 負荷をかけずにサイズを調べる

    find¶

    • findした結果を一気にmv
    • ある日時より後に更新されたファイルを探したい
    • ある日時以前に更新されたファイルを探したい
    • ある期間の間に更新されたファイルを探したい

    git¶

    • HEADにマージしたブランチを削除したい
    • addしたファイルのdiffを確認する(commit前）
    • gitで最初にやる基本設定
    • masterの内容を引き継がずにブランチを作成したい(ドキュメント管理専用ブランチ作成の場合など)
    • すべての（ローカル・リモート） ブランチを表示する
    • まだリモートにpushしてないローカルコミットを確認する
    • コミットしたファイルのdiffを見たい
    • コミットをやり直したい（あの時に戻りたい）
    • ブランチを強制削除したい
    • リモートブランチを削除したい
    • ローカルの変更を一時退避して、先にpullを取り込んでから追加したい
    • 特定のコミットだけ戻したい
    • 特定のコミットだけをやり直したい（取り消し線のイメージ）
    • 特定のバージョン（タグ）に切り替えたい

    grep¶

    • grepした結果を1ファイルにまとめたい

    investigate¶

    • Linuxでプロセスが何のポートを使っているか？
    • Linuxパフォーマンス調査コマンド

    less¶

    • lessでよく使う操作

    ln¶

    • シンボリック先の実パスを取得

    ls¶

    • タイムスタンプ順で表示したい

    mac¶

    • MacのFinderでカラム幅を保存する
    • MacのtopでCPU順のソート
    • MacでコマンドでZIP圧縮する

    oracle¶

    • Oracleでユーザー作成とはじめにつける権限

    shell¶

    • あるディレクトリ配下のファイル数をカウントする

    tmux¶

    • tmuxでよく使う操作

    vim-plugin/vimfiler¶

    • $HOMEに移動
    • helpを開く
    • vimエクスプローラーをvimfilerにする
    • オレオレキー設定
    • カレントディレクトリを開く（cdする）
    • 隠しファイル(.xxxx)を表示する

    vim¶

    • ,を改行に置換する
    • vimでdiffする
    • タブを開く・移動する
    • パスリンクをダイレクトに開く
    • リロード
    • 分割したウィンドウを最大化する
    • 現在のバッファを2画面でdiffする
    • 画面を横に分割する
    • 画面を縦に分割する
    • 空白を除く
    • 開いている（buffer）ファイルに移動
    • 開いている（buffer）ファイル一覧

    watch¶

    • 差分をハイライトする ヘッダーは非表示で
